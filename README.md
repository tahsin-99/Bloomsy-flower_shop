# Bloomsy - Online Flower Shop

Bloomsy is a modern, responsive online flower shop built with **Next.js**, **Tailwind CSS**, and **DaisyUI**. Users can browse flowers, view product details, and manage their account with a login/logout system. [Admin features will include  with adding new flowers]  (with toast notifications for success).
